# Helium 10

> Amazon选品和关键词研究工具(付费),无公开API

## 快速开始

### 安装
```bash
# Chrome扩展
# 访问 Chrome Web Store 搜索 "Helium 10"

# Web界面
# 访问 helium10.com 并注册
```

### 认证配置
- **方式:** 账号登录(无公开API key)
- **环境变量:** 无
- **获取步骤:**
  1. 访问 helium10.com 注册账户
  2. 选择订阅计划(Platinum/Diamond/Elite)
  3. 安装Chrome扩展(可选)
  4. 登录Web平台使用
- **定价:**
  - Starter: $39/月(基础功能)
  - Platinum: $99/月(完整功能)
  - Diamond: $279/月(高级功能+培训)
  - Elite: $399/月(全功能+私人咨询)
- **验证:**
  - 登录 helium10.com 查看Dashboard

## 使用示例

### Black Box - 产品研究
```
1. 登录 Helium 10 → 打开 Black Box
2. 设置筛选条件:
   - 类目: Electronics > Headphones
   - 月销量: 300-3000
   - 价格: $20-$50
   - 评价数: <500
   - 评分: >4.0
3. 点击 Search
4. 查看符合条件的产品列表
5. 分析:
   - 销量趋势
   - 竞争程度
   - 利润空间
   - 评价质量
```

### Cerebro - 反查竞品关键词
```
1. 打开 Cerebro
2. 输入竞品ASIN(可输入多个):
   - B08N5WRWNW
   - B09JQL3NVT
   - B0BDJ8L3LY
3. 选择市场: Amazon.com
4. 点击 Get Keywords
5. 查看结果:
   - 共同关键词(所有ASIN都在用的)
   - 搜索量
   - 竞争度
   - CPC价格
6. 导出关键词列表用于优化自己的listing
```

### Magnet - 关键词研究
```
1. 打开 Magnet
2. 输入种子关键词: "bluetooth headphones"
3. 设置筛选:
   - 最低搜索量: 500
   - 最高竞争度: 50
4. 查看相关关键词列表
5. 分析:
   - 长尾关键词机会
   - 搜索量vs竞争度
   - Magnet IQ Score(综合评分)
6. 添加到关键词列表
```

### Frankenstein - 关键词处理
```
1. 打开 Frankenstein
2. 粘贴从Cerebro/Magnet导出的关键词
3. 选择操作:
   - 去重
   - 删除否定词
   - 按搜索量排序
   - 提取词根
4. 处理后生成精简的关键词列表
5. 复制用于listing标题和后台关键词
```

### Scribbles - Listing优化
```
1. 打开 Scribbles
2. 导入关键词列表
3. 编写Listing:
   - 标题(200字符内)
   - 五点描述
   - 后台Search Terms
4. 实时查看:
   - 关键词覆盖率
   - 字符数统计
   - 优化建议
5. 确保所有重要关键词被使用
```

### Profits - 利润追踪
```
1. 打开 Profits Dashboard
2. 查看指标:
   - 总销售额
   - 总利润
   - 利润率
   - 广告ACOS
   - 退款金额
3. 按产品查看详细:
   - 每个SKU的盈利情况
   - 成本结构(COGS/FBA/广告)
   - 趋势图表
```

### Alerts - 监控设置
```
1. 打开 Alerts
2. 设置监控规则:
   - Hijacker Alert(跟卖监控)
   - Listing Change Alert(listing被篡改)
   - Inventory Alert(库存过低)
   - Review Alert(新差评)
   - Buy Box Alert(丢失购物车)
3. 配置通知方式(邮件/短信)
4. 实时接收异常通知
```

### Chrome扩展使用
```
1. 在Amazon产品页面
2. 点击Helium 10扩展图标
3. 实时查看:
   - X-Ray: 页面所有产品的销量/收入估算
   - ASIN Grabber: 批量提取ASIN
   - Review Downloader: 下载评价
   - Inventory Levels: 库存水平
4. 快速评估产品机会
```
