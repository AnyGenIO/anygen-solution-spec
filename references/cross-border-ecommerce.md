# 跨境电商 — 工具清单

> 详细文档：`read("references/docs-cross-border-ecommerce/<tool-id>.md")`

## 店铺管理

| 工具 | tool-id | 用途 | 安装方式 |
|------|---------|------|---------|
| Shopify CLI | shopify-cli | 官方CLI，管理产品/订单/主题 | `npm install -g @shopify/cli` |
| Shopify MCP Server | shopify-mcp | 官方MCP，AI Agent直接操作店铺 | `npx @shopify/dev-mcp@latest` |
| Amazon SP-API | amazon-sp-api | 亚马逊卖家API，listing/订单/库存 | `pip install python-amazon-sp-api` |
| TikTok Shop API | tiktok-shop-api | TikTok小店，产品/订单/物流 | HTTP API |
| 速卖通 Open Platform | aliexpress-api | 速卖通官方API | HTTP API |
