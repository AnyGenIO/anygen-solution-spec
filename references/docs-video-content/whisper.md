# Whisper (OpenAI)

> 语音转字幕/转写，支持多语言，本地可跑

## 快速开始

### 安装
```bash
pip install openai-whisper
```

### 认证配置

无需认证，开源工具


## 核心能力

- 语音转文字（98种语言）
- 字幕文件生成（SRT/VTT）
- 时间戳对齐
- 本地离线运行

## 适用场景

- 视频自动加字幕
- 会议录音转文字
- 多语言内容转写

## 限制和注意事项

- 大模型需要GPU（large模型约10GB显存）
- 长音频处理较慢
