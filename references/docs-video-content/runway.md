# Runway

> AI视频生成，文字/图片转视频

## 快速开始

### 安装
```bash
无CLI，REST API调用
```

### 认证配置

- **方式：** `api_key`
- **环境变量：** `RUNWAY_API_KEY`
- **获取方式：** dev.runwayml.com 注册获取API Key

## 核心能力

- 文字生成视频（Text-to-Video）
- 图片生成视频（Image-to-Video）
- 视频风格转换
- 视频扩展（延长时长）

## 适用场景

- 生成创意视频素材
- 产品展示视频

## 限制和注意事项

- 付费使用（按秒计费）
- 生成时间较长
- API有限开放
