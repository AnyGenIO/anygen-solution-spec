# yt-dlp

> 视频素材下载(170k stars)，支持1000+网站

## 快速开始

### 安装
```bash
pip install yt-dlp
```

### 认证配置

无需认证，开源工具


## 核心能力

- 下载YouTube/Bilibili/TikTok等1000+网站视频
- 选择画质/格式
- 下载字幕
- 批量下载
- 提取音频

## 适用场景

- 下载竞品视频素材分析
- 下载参考内容

## 限制和注意事项

- 部分网站可能需要登录Cookie
- 注意版权合规
