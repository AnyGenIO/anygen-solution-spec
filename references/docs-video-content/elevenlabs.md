# ElevenLabs

> AI配音/TTS，支持多语言多音色，效果接近真人

## 快速开始

### 安装
```bash
pip install elevenlabs
```

### 认证配置

- **方式：** `api_key`
- **环境变量：** `ELEVENLABS_API_KEY`
- **获取方式：** elevenlabs.io → Profile → API Key

## 核心能力

- 文字转语音（29种语言）
- 语音克隆
- 实时流式TTS
- 音色定制

## 适用场景

- 视频AI配音
- 多语言内容本地化
- 有声内容生成

## 限制和注意事项

- 免费版每月1万字符
- 付费$5-99/月
