# MoneyPrinterTurbo

> 一键生成短视频(54k stars)，自动脚本+素材+配音+字幕

## 快速开始

### 安装
```bash
git clone https://github.com/harry0703/MoneyPrinterTurbo
pip install -r requirements.txt
```

### 认证配置

无需认证，开源工具


## 核心能力

- 自动生成视频脚本
- 自动搜索匹配素材
- 自动AI配音
- 自动添加字幕
- 一键输出成品视频

## 适用场景

- 批量生成带货/种草短视频
- 自媒体内容批量生产

## 限制和注意事项

- 需要配置LLM API Key（用于生成脚本）
- 素材质量取决于搜索源
