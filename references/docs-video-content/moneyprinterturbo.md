# MoneyPrinterTurbo

> AI 自动化短视频生成工具，一键生成带货/解说视频

## 用途
- AI 生成视频文案（OpenAI/通义千问/GLM）
- 自动匹配视频素材（Pexels/Pixabay）
- AI 配音（Azure TTS/ElevenLabs/Edge-TTS）
- 自动字幕生成和渲染
- 批量生成视频，支持多语言
- 自定义文案和素材

## 适用场景
- 短视频带货（抖音/快手/视频号）
- 知识解说和产品介绍视频
- 热点资讯快剪和读书/电影解说
- 教程类视频和营销推广视频
- 旅游/美食 vlog 和励志/情感短视频

## 注意事项
- 需要 API Keys：OpenAI（文案生成）、Azure TTS/Edge-TTS（配音）、Pexels（素材，免费）
- 必须安装 ImageMagick（字幕渲染依赖）
- OpenAI API 约 $0.002-$0.01/视频，Azure TTS 免费额度 500,000 字符/月
- Pexels API 免费但有速率限制（200 请求/小时）
- AI 生成文案需人工审核质量
- 完全开源免费（MIT 许可证）
