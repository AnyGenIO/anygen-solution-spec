# FFmpeg

> 开源视频处理工具，转码、剪辑、合并、滤镜

## 用途
- 视频格式转换（MP4/MOV/AVI/WebM/MKV 等）
- 视频压缩和码率调整
- 视频剪辑、合并、音视频分离
- 添加水印、字幕、滤镜
- 帧提取和截图
- 流媒体推流、硬件加速编码

## 适用场景
- 视频格式批量转换和压缩优化
- 自动化视频剪辑和素材预处理
- 提取视频帧用于 AI 训练或分析
- 音视频同步调整和流媒体推流
- 社交媒体视频上传前的质量优化

## 注意事项
- 无需 API Key，本地运行完全免费
- 编码速度取决于硬件性能和参数配置
- CRF 值范围 0-51（推荐 18-28，越低质量越高）
- 使用 `-c copy` 快速处理但不能改变编码参数
- 硬件加速需要兼容显卡（NVIDIA/AMD/Intel）
