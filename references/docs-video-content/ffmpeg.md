# FFmpeg

> 视频处理基础库，剪辑/转码/合并/提取音频

## 快速开始

### 安装
```bash
apt install ffmpeg  # 或 brew install ffmpeg
```

### 认证配置

无需认证，开源工具


## 核心能力

- 视频转码（格式转换）
- 视频剪辑（裁剪/拼接/分割）
- 音频提取和替换
- 字幕硬烧
- 视频压缩
- 批量处理

## 适用场景

- 视频格式转换
- 提取音频轨道
- 批量压缩视频

## 限制和注意事项

- 纯命令行，无GUI
- 参数复杂，学习曲线陡
